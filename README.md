# AoiStitcher v1.3.1

**极简、高效、优雅的长图拼接工具。**

---

## 💭 开发者感悟 (v1.3.1 Notes)

在这个版本的开发过程中，我最大的感悟就是发现 **Python** 真的非常神奇。

它竟然能做到”**一次编写，多端运行**“。同一套代码，可以在不同的操作系统上完美运行。这种跨平台的能力，让原本复杂的开发工作变得简单而有趣，也促成了这个版本的快速落地。

---

## ✨ 核心特性

AoiStitcher 延续了 v1.3 的底层架构，专注于为摄影师提供最流畅的拼图体验：

* **⚡️ 丝滑交互**：采用磁吸式拖拽排序，照片自动避让，操作还算流畅吧。
* **💧 自由水印**：支持 Logo 比例缩放及位置的像素级微调，满足严苛的排版需求。
* **🧠 智能记忆**：自动记录导入、导出及 Logo 路径，彻底告别重复点击文件夹的繁琐。
* **👁 沉浸视觉**：支持黑/白画布一键切换。

---

## 📥 下载与运行 (无需配置)

为了让大家更方便地使用，本工具已打包为可执行程序，**无需安装 Python 或配置任何代码环境**。

* **Windows / macOS 用户**：直接获取对应平台的安装包/可执行文件，双击即可直接运行。
* **开箱即用**：程序已内置所有运行所需的依赖，下载即用。

---
