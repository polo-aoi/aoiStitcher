**📸 aoi拼图 (aoiStitcher) v1.2**

💡 开发初衷：一个摄影师的效率自救
我是 Polo Aoi (ポロ蒼)。我的主业是在化工厂负责污水处理，长年累月地在昼夜颠倒的倒班生活中寻找平衡。同时，我运行着自己的摄影工作室，是一名婚礼摄影师。
每次拍摄结束，最折磨我的不是修图，而是那最后半小时到四小时的机械拼图排版。我也考虑过像“像素苏打”这样的付费软件，但作为每一分钱都靠倒班和奔波攒出来的兼职摄影师，我确实“不舍得”那笔订阅费。
于是，在 AI 的帮助下，我这个零基础的编程小白，为自己（也为所有追求效率的同行）“搓”出了这个专为 Mac 设计的极简拼图工具。
✨ 核心功能
 * 🚀 极速启动：针对 Mac M 系列芯片深度优化，采用文件夹模式封装，实现秒开，拒绝等待。
 * 🧠 参数记忆：软件会自动记住你上次设置的宽度、间距、Logo 路径以及导出位置，下次打开直接开干。
 * 🎨 极简 UI：采用 iOS 风格的深色模式设计，扁平化视觉，操作直观不刺眼。
 * 🖼️ 拖拽即用：支持从 Finder 直接拖拽多张照片进行自动对齐拼接。
 * 🏷️ 智能 Logo：完美支持 PNG，PSD 格式的 Logo 拖拽，自动居中适配底部区域。
📦 如何下载与安装
 * 下载：点击页面右侧的 [Releases] 区域，下载最新的 aoi拼图_v1.2_正式版.dmg 文件。
 * 安装：双击打开 DMG，将 aoi 拼图 图标拖拽到右侧的 Applications 文件夹中。
 * 首次运行（重要）：
   * 由于未在苹果官方备案，首次打开请右键点击软件图标，选择“打开”。
   * 在弹出的系统安全提示中，再次点击“打开”即可正常使用。
🛠️ 技术实现
 * 语言：Python 3.14
 * 图形库：Tkinter + tkmacosx (解决 Mac 按钮变色问题)
 * 图片处理：Pillow (PIL)
 * 拖拽支持：TkinterDnD2
