
# AoiStitcher v1.3 📸

**AoiStitcher** 是一款专为摄影师设计的极简风长图拼接工具。它采用了 iOS 风格的 UI 交互，支持磁吸式拖拽排序、高精度水印微调以及全自动路径记忆，旨在为样片展示和社交媒体分享提供最高效的后期工作流。

---

## ✨ 新特性 (v1.3 更新日志)

相比于之前的 v1.2 版本，v1.3 经历了重大的底层重构和体验升级：

### 1. 丝滑的交互体验
- **磁吸感应排序**：拖拽照片时，周围照片会实时自动滑开避让，实现“所见即所得”的排序体验。
- **性能飞跃**：引入异步重绘机制与二级预览缓存，即使处理几十张高像素原图也能保持 UI 流畅。

### 2. 自由的水印系统
- **三维微调面板**：新增滑块控制，支持 **Logo 比例 (%)**、**水平位置**、**垂直位置** 的像素级偏移。
- **水印库记忆**：自动记录最近使用的 6 个 Logo 文件，支持右键快捷菜单一键切换。

### 3. 智能路径记忆
- **全场景记录**：系统会自动保存上次 **导入照片**、**上传 Logo** 以及 **导出成品** 的文件夹路径，大幅减少重复点击。
- **实时路径展示**：侧边栏底部动态显示当前输出目录，清晰掌控文件去向。

### 4. 视觉与交互优化
- **iOS 极简深色主题**：针对摄影师审美设计的纯黑工作区，减少视觉干扰。
- **双色画布预览**：支持一键切换黑/白背景，完美匹配不同风格的照片。
- **快捷键支持**：选中照片后支持 `Delete` 或 `Backspace` 一键删除。
